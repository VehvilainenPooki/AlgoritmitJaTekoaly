# Kohinanpoistotyökalu
Tarkoituksena on toteuttaa työkalu, jolle voi antaa äänitiedostoja. Työkalu antaa vaihtoehtoja siitä kuinka voimakkaan kohinanpoiston tiedostolle suoritetaan.
